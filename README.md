# Inventory Sort Fixed

Minetest модуль для сортування інвентаря з фіксованими слотами (priority slots).

## Команди
- `/sort` — відкриває інвентар з кнопкою сортування.

## Особливості
- Перші 8 слотів — priority slots (завжди заповнені зі стека main).  
- Кнопка сортує залишок інвентаря за алфавітом.  
- Сумісно з Minetest 5.x та formspec версії 4.  

## Ліцензія
GPL v3 — можна вільно використовувати і модифікувати з вказанням автора.
